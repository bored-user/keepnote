Skip()

Takes an integer as argument indicating how many numbers to skip. Will return a collection starting from item n - where n is the given argument.

If n is greater than the size of the collection, an empty collection is returned. If it's 0 or smaller, the whole collection will be returned.

using System;

class Program
{
	static void Main()
	{
		int[] nums = new int[] { 1, 2, 3 };
		Console.WriteLine(nums.Length); // 3
		Console.WriteLine(nums.Skip(2).ToArray().Lenght); // 1
	}
}
