Boxing and Unboxing

Boxing is the act of storing a primitive value, say an integer, on an object variable. That's possible because object is the base class on C#.

Unboxing is the practice of converting a primitive value, say an integer, stored on an object to the actual primitive type again.

What happens under the hood is that C# stores a pointer to an integer on the stack and the actual integer on the heap - it treats it like a reference type.

Boxing and unboxing negatively affects performance.
