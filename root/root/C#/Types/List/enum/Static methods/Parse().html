Parse()

The Enum.Parse() method takes a typeof() output (representing the reference to the enum), a string as the enum's key and a optional boolean indicating if the method is supposed to ignore case (defaults to false). Returns the value of the given key.

class Program
{
	enum Colors
	{
		Red = 255,
		Green = 255,
		Blue = 255
	}
	
	static void Main()
	{
		Console.WriteLine(Enum.Parse(typeof(Colors), "rEd", true));
		Console.WriteLine(Enum.Parse(typeof(Colors), "Red", false));
		/*
			#=> 255
			#=> 255
		*/
	}
}
