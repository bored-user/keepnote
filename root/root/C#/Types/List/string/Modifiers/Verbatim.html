Verbatim

Verbatim strings are useful when one needs to use special characters, like backslashes. Usually, \n would be interpreted as a new line. On a verbatim string, however, it isn't.

Verbatim strings are prefixed with an @ sign:

string path = "c:\\\Users";
string path_verb = @"c:\\Users";

// Note that the first string needs an extra backslash as an escape character.

 

One can create new lines on verbatim strings by literally skipping a line.
