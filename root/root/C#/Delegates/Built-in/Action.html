Action

An Action is built-in delegate. It's very similar to System.Func. The only difference is the fact that Action returns void while System.Func has a return value.

Finally, an Action may or may not use generics to define arguments.

 

General syntax:

class Program
{
	static void Main()
	{
		Action a = delegate() { Console.WriteLine("no args"); };
		Action<string> b = delegate(string s) { Console.WriteLine(s); };
		
		a();
		b("with args as well!");
		
		/*
			#=> "no args"
			#=> "with args as well!"
		*/
	}
}
