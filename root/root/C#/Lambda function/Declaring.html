Declaring

All lambda functions follow a basic declaration path: the argument list, the arrow sign (=>) and the implicit return statement.

However, when a lambda function is more than one line long, it's possible to open a curly braces block after the arrow sign - but the return statement needs to be implicit.

Here's an example:

class Program
{
	static void Main()
	{
		Func<ulong, ulong> fact1 = n => n <= 1 ? 1 : n * fact1(n - 1);
		Func<ulong, ulong> fact2 = n =>
		{
			if (n <= 1)
				return 1;
			else
				return n * fact2(n - 1);
		}
		
		Console.WriteLine($"Fact1: {fact1(5)}\nFact2: {fact2(5)}");
		
		/*
			In this example 1, I've used a ternary statement to decide what to return, with an implicit return statement.
			
			It's, however, possible to explict that return statement and use a curly brace block, like shown in example 2.
			
			Note that both output 120.
		*/
		
	}
}
