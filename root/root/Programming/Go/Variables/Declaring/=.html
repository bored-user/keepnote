=

:=

The gopher (aka short variable declaration operator) operator can only be used inside a closure. It cannot be used to declare global variables. It should always (when possible) be used. Has type-inference & must declare, at least, one variable.

package main

import "fmt"

func main() {
	x := 10
	x, y := 20, 10
	
	fmt.Println(x, y)
	// 20 10
}
