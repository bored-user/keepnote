$group

$group: group results according to specified parameters. Needs to take an accumulator (_id) and any other optional parameter that can be, for example, incremented.

Syntax: { $group: { _id: '$tags', count: { $sum: 1 } } } // _id = accumulator and count is opt (will be incremented at // each loop)

Result: [ { '_id': 'tag_1', count: 1 }, { _id: 'tag_2', count: 2 } ]
