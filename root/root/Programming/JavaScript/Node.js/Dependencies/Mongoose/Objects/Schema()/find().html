find()

find() is a function that can either take one or no arguments.
If no argument is given, it returns an array of objects where each object is an item on the connected database.

If one chooses to pass in the argument, it should be an object where the key is a field on the database schema and the value is the searched value.

Example:
model.find({ field1: 'value1' });

Observation: one may use model.find() to search for an item in an array field:

user_1 {
	friends: ['friend1', 'friend2', 'friend3'];
}



mongoose.Schema({}).find({ friends: 'friend1' }) // will return user_1
