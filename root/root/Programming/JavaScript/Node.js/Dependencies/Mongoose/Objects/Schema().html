Schema()

mongoose.Schema() is a function to create new database schemas.
It takes an object as argument, containing the field name and type of data that it'll accept.

One can, instead of giving the type of data as the object values, pass in another object containing advanced options.

Example:

 

const schema = new mongoose.Schema({
	field1: String,
	field2: Number,
	field3: [],
	field4: {
		required: true
		type: String
		trim: true
	},
	field5: [String] // array of strings
});
