Express

Express is the "most have" dependency used to create the server, routes, etc.

 

In order to send data from client to server as a POST call, one must specifiy
express().use(express.json()) and express().use(express.urlencoded({ extended: true }));

Also, on the fetch/ XMLHttpRequest, one must specify the header 'Content-Type' (to 'application/json').

 

Static directories are directories treated as normal directories: the user can access the files in it, without the need to specify routes.
One naming convention of static directories is the use of the 'public' folder name.
In order to set it up, one must use express().use(express.static('dir_name')). Then, the files inside it can be accessed like normal files.
