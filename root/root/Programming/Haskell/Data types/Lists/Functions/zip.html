zip

Takes two lists and pairs them up in tuples: zip [1..5] ["one", “two”, “three”, “four”, “five”] returns [(1, “one”), (2, “two”), (3, “three”), (4, “four”), (5, “five”)].

If one of the lists is longer than the other, it is chopped off to match the smaller. For instance: zip [1..] ["a", “b”, “c”] returns [(1, “a”), (2, “b”), (3, “c”)].
