Multiple dispatch

There's also that multiple dispatch thing, like Julia: one can define multiple versions of a function, taking args from different types and returning different types. Swift will automatically decide what to call.

func concat_sum(a1: Int, a2: Int) -> Int {
	n1 + n2
}

func concat_sum(a1: String, a2: String) -> String {
	s1 + s2
}

concat_sum(a1: 1, a2: 2) // 3
concat_sum(a1: "1", a2: "2") // 12
