Functions

Functions are declared using the func keyword. One needs to namely specify (if any) the function's arguments and their possible aliases.

A function may be a void, which is a function that return nothing.

func sum(a: Int, b: Int) -> Int {
	a + b // It takes 2 Ints, `a` and `b` and returns (keyword migth be omitted) their sum.
}

func my_void() {
	// Returns nothing. Note the absence of `-> [type]`.
}

func aliases(n1 alias1: String, n2 alias2: String) -> String {
	return alias1 + alias2
}
