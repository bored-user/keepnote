const

The const modifier generates a constant. A constant - as the name suggests - cannot be modified. For instance, those operations will result in error:

#include <iostream>
#include <string>

int main() {
	const int nums[] { 1, 3, 5, 7, 9 };
	nums[0] = 2;
	
	const std::string name = "Joao";
	name = "Vaz";

	return 0;
}
