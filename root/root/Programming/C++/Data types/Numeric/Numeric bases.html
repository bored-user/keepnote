Numeric bases

One can represent numbers in C++ in different bases:

  • decimal: int n = 10;
  • binary (prefixed 0b): int n = 0b111;
  • octal (prefixed 0): int n = 017;
  • hexadecimal (prefixed 0x): int n = 0x0F;

Note that all of those numbers will be evaluated as decimal; however, it's possible to store other bases.
