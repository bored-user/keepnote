(De)Referencing

One may get the address of a variable (referencing) using int* a_ptr = &a - where a is a previously defined variable of type integer. Threfore, &a gets the address of a.

Then, one may dereference the pointer and get the value back using *a_ptr. Therefore, *a_ptr gets the value stored in the address represented by a_ptr.

#include <iostream>

int main() {
        int a = 10;
        int* a_ptr = &a;

        std::cout << a_ptr << std::endl;   // 0x...
        std::cout << a << std::endl;	   // 10

        *a_ptr = 12;					   // changing a
        std::cout << a_ptr << std::endl;   // 0x...
        std::cout << a << std::endl;	   // 12

        return 0;
}
