Dynamic memory allocation

One can allocate memory on the heap to be used (only by) the current script using the new keyword. Afterwards, one must run delete ptr; to return the memory to the operating system and it's a good practice to also run ptr = nullptr; in order to be sure no one uses it again without redefining by accident

#include <iostream>

int main() {
	int* a_ptr = new int(69);
	// using a_ptr

	delete a_ptr;
	a_ptr = nullptr;

	return 0;
}
