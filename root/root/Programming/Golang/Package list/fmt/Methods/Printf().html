Printf()

Used to get information on the given value.

For instance, %T returns the value type, %v returns the value itself.

Example:

package main

import "fmt"

func main() {
	x := 1
	y := "hello"
	
	fmt.Printf("%v\t%T", x, x)
	fmt.Printf("%v\t%T", y, y)
	
	//#=> 1 int
	//#=> hello string	
}

 

Each other argument will be used as base for a formatter parameter.
