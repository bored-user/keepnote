make()

The make() function creates a slice with of type T (first argument) with L length (2nd) and C capacity (3rd). It improves the performance of slices, since it allows Go to only recreate the array when the capacity is reached.

For instance;

package main

func main() {
	// Creates a slice which underlying array is 3 slots long.
	// The length of the actual slice is 1 (only slice[0] is allowed)
	slice := make([]int, 1, 3)
	
	// Filling slice's length
	slice = append(slice, 0)
	slice = append(slice, 0)
	
	// Now, a new array will be created.
	// It'll have a length (aka slice's capacity) of 6 (it doubles)
	// And the slice can keep on increasing
	slice = append(slice, 0)
}
