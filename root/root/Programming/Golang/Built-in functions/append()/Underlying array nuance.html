Underlying array nuance

Note that when assigning the result of append() to a new array, the underlying type of the old array is modified, meaning that the old array is modified (illustrated below). Therefore, it's a best practice to use other methods to expand an array (when dealing with +1 array), like a for loop.

package main

import (
	"fmt"
)

func main() {
	slice1 := []int{1, 2, 3, 4, 5}
	fmt.Println(slice1)
	// [ 1 2 3 4 5 ]
	
	slice2 := append(primeiroslice[:2], primeiroslice[4:]...)
	fmt.Println(slice2)
	// [ 1 2 5 ]

	fmt.Println(slice1)
	// [ 1 2 5 4 5 ]
	// Note how `slice1` was modified even though nothing changed its value directly
}
