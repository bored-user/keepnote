Spread-like operator

Just like in Julia or JavaScript, it's possible to use a spread-like operator in Go.

The spread operator is used to spread the contents of an array/ slice as individual arguments to a function.

Taking the previous note sum function example, one could do:

slice := []int { 1, 2, 3, 4, 5 }
sum(slice...)	// 15

 

This is also useful to join slices together.
