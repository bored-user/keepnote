Anonymous functions

An anonymous function may be assigned to a variable or just called immediately:

x := func() string { return "aa" }
x()		// aa

y := func(a int, b int) int { return a + b }(10, 20)
// y == 30

Therefore, functions that return functions are also allowed - as well as closures:

func a() (func() int) {
	return func() {
		return 1 + 2
	}
}

a()()	// 3
