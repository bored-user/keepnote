Return value

When a function returns something (that isn't void), one must specify the data type - which is done using a space character, just like varvariables.

package main

import "fmt"

func str() string {
	return "string here!"
}

func integer() int {
	return 0
}

func boolean() bool {
	return true
}

func main() {
	fmt.Println(str())
	fmt.Println(integer())
	fmt.Println(boolean())	
}
