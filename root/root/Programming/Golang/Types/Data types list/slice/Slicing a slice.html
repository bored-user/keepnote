Slicing a slice

One can slice a slice using a like-Python notation, using square brackets and colons: slice[x:y]. The x index is included but the y is excluded.

Note that this will return another slice with the proper elements (even if there's only one element).

package main

import "fmt"

func main() {
	slice := []int { 0, 1, 2, 3, 4 }
	
	fmt.Pritnln(slice[:])		// same as fmt.Println(slice)
	fmt.Println(slice[0:2])		// [0 1]
	fmt.Println(slice[:3])		// [0 1 2]
	fmt.Println(slice[3:])		// [3 4]
	
}
