Multi-dimensional slices

Note that it's possible to create multi-dimensional slices like so:

package main

func main() {
	slice2d := [][]int{
		[]int{
			1,2,3,
		},
		[]int{
			4,5,6,
		},
	}
	
	// slice2d[1][1] == 5
}

Be aware that there needs to be a comma after the []int slice's last element and after the [][]int slice's last element (unless they are declared in a single line like so [][]int{ []int { 1,2,3 }, []int { 4,5,6 } }).
