Check existance of entries

One can use the map_variable[TKey] notation to retrieve the value held by a map. However, note that the Go language will return the default type value (aka zero value) if there's no entry in that TKey. One can verify if there really isn't an entry using the comma ok idiom like so:

package main

import "fmt"

func main() {
	mymap := map[int]string {
		1: "a",
		2: "b",
	}
	
	if v, ok := mymap[99]; !ok {
		fmt.Println("mymap doesn't contain key 99")
	} else {
		fmt.Println("mymap contains key 99")
	}
}
