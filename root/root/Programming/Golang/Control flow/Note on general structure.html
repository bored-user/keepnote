Note on general structure

Usually, in programming languages like JavaScript, Java, C#, C++, etc, if, for, while, etc blocks' conditions are written enclosed in parentheses, like so:

// C#
for(int i = 1; i <= 5; i++)
	System.Console.WriteLine(i);
	
// JS
for(let i = 1; i <= 5; i++)
	console.log(i);
	
// Java
if(i <= 5)
	System.out.printLn("i is less than or equal to 5");

 

In Go, however, there's no parentheses! One simply writes the conditions of the loop or if block:

package main

import "fmt"

func main() {
	for i := 1; i <= 5; i++ {
		if i == 1 {
			fmt.Println("Starting the loop...")
		}
	
		fmt.printLn(i)
	
		if i == 5 {
			fmt.Println("Ending the loop...")
		}
	}
}

 

Also, note that, in the languages mentioned above, when there's only one instruction on the conditional or loop structure, one can omit the curly braces. In Go, however, there always has to be a set of curly braces.
