Declaring

One can declare a variable using the var keyword or the gopher operator (:=). The basic differences are:

  1.  
  2. The gopher operator must declare at least one variable.
  3. The gopher variable has type inference.

 

Explanation of number 2 is illustrated bellow:

package main

import "fmt"

func main() {
	x := 10
	fmt.Println(x)
	
	// x := 20  This will fail because the gopher isn't declaring vars, only assigning
	x, y := 20, 10 // This works because it's also declaring `y`
	
	fmt.Println(x, y)
}
