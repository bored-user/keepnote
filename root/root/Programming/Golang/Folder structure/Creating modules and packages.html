Creating modules and packages

To create a module, one simply needs to create a new folder inside $GOPATH/src/[site]/[user] (like $GOPATH/src/github.com/bored-user/project1) and run go mod init [site]/[user]/[module] - in this case, go mod init github.com/bored-user/project1. The command will create a go.mod file, indicating informations on the module.

To create a package, simply create a folder with the package name and a go script with the tool name; for instance $GOPATH/src/github.com/bored-user/project1/pkg1/tool.go. Inside it, there must be the package declaration matching the folder name (package pkg1) and at least one exported identifier.

To import it, simply use the import directive, pointing to [site]/[user]/[module]/[package], like so import “github.com/bored-user/project1/pkg1”.
