using Static Directive

The using static directive works much like the using ("dynamic", I suppose) directive.

The only difference is that the using directive only supports namespaces - while the using static directive supports classes & namespaces.

An example:

using static System.Console;
using static System.Math;

class Program
{
	static void Main()
	{
		WriteLine(Sqrt(4)); // #=> 2
	}
}
