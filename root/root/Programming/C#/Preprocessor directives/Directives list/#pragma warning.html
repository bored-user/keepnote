#pragma warning

Disables or enables some warning messages. One must specify a comma separated list of warnings (the CS prefix is optional).

In order to disable a warning, one can use #pragma warning disable 414, CS3021 - which disables warnings CS414 and CS3021. It's also possible to use #pragma warning disable which disables all warnings.

To enable warnings, #pragma warning restore 414, CS3021, which enables CS414 and CS3021. Similarly, #pragma warning restore enables all warnings.
