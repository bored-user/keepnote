#if

The #if directive makes the compiler compile the block only if the given symbol is true. Note that it's possible to use ==, &&, || and !. It normally comes with a #endif, #elif or #else directive. Same as an if block. Note that DEBUG on the example below is a symbol defined using #define.

// same as #if (DEBUG == true)
#if DEBUG
	Console.WriteLine("on debug mode");
#endif
