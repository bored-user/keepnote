Overflow

By default, C# won't check for data overflows. Which means that this happens:

byte num = 255;
num = num + 1;
System.Console.WriteLine(byte); // 0

 

To prevent this kind of behaviour, one can enclose that dangerous code on a checked block:

checked
{
	byte num = 255;
	num = num + 1;
	System.Console.WriteLine(byte); // Exception
}

 

Using try/catch:

try
{
	checked
	{
		byte num = 255;
		num = num + 1;
		System.Console.WriteLine(byte); // Exception
	}
}
catch (Exception err)
{
	System.Console.WriteLine(err);
}
