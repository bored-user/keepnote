Creating

To create an array, one can use the T[] = new T[n] syntax, where T is the data type and n is the array length. If not specified, the array will be created with default values.

int[] ages = new int[5];
// [0, 0, 0, 0, 0]

 

It's also possible to pass in a block with values, and even omit the n:

int[] ages = new int[5] { 1, 2, 3, 4, 5 };
int[] @ages = new int[] { 1, 2, 3, 4, 5 };

 

Finally, one can also use the nullable value types on an array:

using System;

bool?[] nulls = new bool?[3];
Console.WriteLine(nulls); // => [null, null, null]
