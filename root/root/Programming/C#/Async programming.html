Async programming

C# is a synchronous programming language by default. Meaning that it stops at execution of processes that take time (a web request, for instance). It's possible to program asynchronously using async methods and the await keyword - just like in JavaScript:

class Program
{
	static async Task AsyncTask()
	{
		// something that takes time to execute
	}
	
	static async void Main()
	{
		var res = await AsyncTask();
	}
}

 

Note that all synchronous methods have an async version, which signatureis MethodAsyncTask. The method can return either void or a System.Threading.Tasks.Task object - for which there's a generics version. Finally, note that only async methods can use the await keyword.

Be aware that it's also possible to execute a TaskAsync function, store its result on a variable and only later use await on it. The System.Threading.Tasks.Task object is essentially a JavaScript's Promise object.
