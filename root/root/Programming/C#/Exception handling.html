Exception handling

Error handling can be done using the try and catch blocks.

using System;

try
{
	int n = 10 / 0;
}
catch(Exception exc)
{
	Console.WriteLine(exc);
	// #=> Division by constant zero
}

 

Note that the exception is from the type Exception. This encapsulates all exceptions, but one can specify an exception for each catch block.

Finally, note that there's a finally block, which gets executed independently if there were or not errors on the script.

When dealing with files, it's necessary to make a try {}, catch {} and finally {} blocks - the latter executes the Dispose() method, closing the stream. All of this can be replaced by a using statement.

Note that it's also possible to create a custom exception by creating a class that inherits from the Exception class.
