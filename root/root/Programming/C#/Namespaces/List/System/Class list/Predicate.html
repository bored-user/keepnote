Predicate

A Predicate is a special type of delegate, just like Action and Func - which returns a boolean. Note that, usually, a Predicate is defined as a lambda expression since it's easier to access local scope variables.

See the example below:

class Program
{
	static void Main()
	{
		int[] nums = new int[] { 1, 2, 3, 4, 5 };
		Predicate<int> IsEven = n => n % 2 == 0;
		
		Array.ForEach(Array.FindAll(nums, IsEven), n => Console.WriteLine(n));
		/*
			#=> 2
			#=> 4
		*/
	}
}
