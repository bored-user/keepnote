Where()

Where() is a method that takes a delegate (a Predicate ) and returns a new collection formed by the items where the given Predicate returned true. It's like JavaScript's or Ruby's filter().

using System;
using System.Linq;

class Program
{
	static void Main()
	{
		int[] nums = new int[] { 1, 2, 3 };
		Console.WriteLine(nums.Where(n => n % 2 == 0).ToArray()[0]); // #=> 2
	}
}
