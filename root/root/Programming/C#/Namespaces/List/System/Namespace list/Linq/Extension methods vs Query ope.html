Extension methods vs Query operators

LINQ's extension methods are the default of method calling in C#. The methods are called on the collections, and receive a lambda expression. They can be chained.

Query operators, on the other hand, are a little bit like SQL. They all start with the from statement and end with select. For instance:

using System.Collections.Generic;
using System.Linq;
using System;

class Book
{
	public string Title { get; set; }
	public double Price { get; set; }
}

class Program
{
	static List<Book> GetBookList() => new List<Book>() { { Title = "C# Guide", Price = 10.5 }, { Title = "Java Guide", Price = 10 }, { Title = "Python Guide", Price = 12 } };

	static void Main()
	{
		var books = GetBookList();
		
		var ExtMethBooks = books
					.Where(b => b.Price < 11)
					.OrderBy(b => b.Title)
					.Select(b => b.Title)
		var QueryOpBooks = 
					from b in books
					where b.Price < 11
					orderby b.Title
					select b.Title
					
		// ExtMethBooks and QueryOpBooks are the same thing
	}
}
