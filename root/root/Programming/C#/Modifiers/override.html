override

The override keywords overrides inherited virtual/ abstract methods/ properties. For instance, to implement a custom ToString() method, it's necessary to declare a public override ToString() method on the class.

abstract class ParentClass
{
	public abstract void MyMeth();
	public abstract int n;
}

class ChildClass
{
	public override void MyMeth() => Console.WriteLine("Overwritten");
	public override int n = 10;
}
