Keywords

Here's the list of constraint keywords:

class MyCustomType
{}

interface IMyCustomInterface
{}
  • where T : IMyCustomInterface - where the type implements IMyCustomInterface
  • where T : MyCustomType - where type is a MyCustomType
  • where T : class  - where the type is a reference type
  • where T : struct - where the type is a value type
  • where T : new() - where the type has a constructor

 

The final syntax is, for instance:

class GenericClass<T> where T : class
{}
