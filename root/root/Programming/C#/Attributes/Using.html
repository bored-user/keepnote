Using

An attribute can be stated on a line above a field or at the same line - before the field declaration.

For instance:

[Obsolete]
class MyObsoleteClass1
{}

[Obsolete] class MyObsoleteClass2
{}

Even though, as mentioned above, it's syntactically allowed to state the attribute at the same line as the field declaration, best practices state that it's better to use the inline version only on properties.

 

Be aware that it's also possible to give arguments to a attribute, calling it just like a function:

[Obsolete("Reason why it's obsolete")]
class MyObsoleteClass3
{}

 

Finally, notice that, by convention, the C# language allows one to shorten its syntax, writing Name instead of NameAttribute (for instance, Obsolete instead of ObsoleteAttribute - which is the actual name of the Obsolete attribute class).
