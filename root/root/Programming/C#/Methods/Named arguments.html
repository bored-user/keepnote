Named arguments

In C#, method arguments can be sent in two ways: positional and named.

Positional arguments are the default way for most programming languages: method A gets string a, int b, string c and, when called, receives, for instance "hello, ", 2, “world!”.

Named arguments allows one to pass in methods in any order - using their labels, like so:

using System;

public void A(string a, int b, string c)
{
	for(int i = 0; i < b; i++)
		Console.WriteLine($"{a}{b}");
}

static void Main()
{
	A(c: "hello, ", b: 2, a: "world!");
	/*
		#=> "hello, world!"
		#=> "hello, world!"
	*/
}
