Embedded Ruby

There are four ways to embed Ruby into HTML:

  1. <% %>
  2. <%= %>
  3. <%= -%>
  4. <%# %>

The first executes Ruby code but don't display anything on the page.

My pretty text <% puts 10 * 10 %>
#=> My pretty text

The second displays the result of the typed code on the page but add a line-break at the end of the input.

My pretty text <%= 10 * 10 %>
#=> My pretty text 100

The third one performs exactly like the second but doesn't display line-breaks.

My pretty text <%= 10 * 10 -%>
#=> My pretty text 100

The last one represents a comment. It won't be interpreted at all.
