Binding class

The binding class is a way to access local variables on the global scope. It creates an anchor on the code, allowing one to access the variables later on.

After the binding method is used, the function's class (where said binding method was called) turns to BindingClass.

def normal_meth
    a = 123
    'my normal return'
end

puts normal_meth.class #=> 'String'


def binding_meth
    a = 123
    binding
    'my normal return'
end

puts normal_meth.class #=> 'BindingClass'
puts binding_meth.eval('a') #=> 123
puts a #=> ERROR
