puts

The puts function is used to log something on the console.

The difference between p and print is that it renders the nil value as a blank space and adds a line break at the end of the printed string.

Before printing, puts tries to convert the data type to a String, calling the to_s method.

puts 'hello, '
puts 'world!'
#=> 'hello, 
#    world!'

puts can also be used to write something to an open file:

f = File.new('file', 'w')
f.puts('hello, world!')
f.close
