Implicit vs Explicit

The implicit block usage is the default one: passing the block outside the function's argument parentheses and using yield.

But there's also the explicit version: listing the block in the function's argument list through the & symbol and the block.call method.

 

def implicit
	yield 'inside implicit!'
end

def explicit(&block) # note the & sign
	block.call 'inside explicit!' # same as yield
end


implicit { |s| puts s } # 'inside implicit!'
explicit { |s| puts s } # 'inside explicit!'
