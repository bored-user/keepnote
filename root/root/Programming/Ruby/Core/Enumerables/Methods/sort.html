sort

The sort method can be used to sort Arrays, Hashes and Enumerables. When sorting string (or number) Arrays or string (or number) key Hashes, the result is pretty obvious: alphabetical order and increasing one. However, when sorting multi-dimensional Arrays/ Hashes, things might get a little complicated.

arr = ['f', 'w', 'l', 'g']
p arr.sort #=> ['f', 'g', 'l', 'w']

hash = {'f' => 1, 'w' => 1, 'l' => 1, 'g' => 1}
p hash.sort #=> [['f', 1], ['g', 1], ['l', 1], ['w', 1]]
