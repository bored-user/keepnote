delete

Removes all instances of a given character from a String and returns (a modiff version of) such string.

str = 'joao'

puts str.delete('a') #=> 'joo'
puts str.delete #=> ERROR

Also works in Hashes:

hash = {'a' => 1, 'b' => 2}
hash.delete('a')
puts hash #=> {'b' => 2}

And on the File object:

File.delete('file', 'w')
File.new('file', 'w').delete #=> ERROR
