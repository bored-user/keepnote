inject

Acts like JavaScript map: takes an accumulator as first argument and a block as “second”.

arr = [1, 2, 3, 4, 5]
sum = arr.inject(0) {|acc, n| acc += n}
prod = arr.inject(0) {|acc, n| acc *= n}

puts sum, prod
#=> 15, 120

 

When no argument is given as accumulator, the method starts at array index 1 and the accumulator turns to the 1st array element (index 0).

arr = [1, 2, 3, 4, 5]
sum = arr.inject {|acc, n| acc += n}
prod = arr.inject {|acc, n| acc *= n}

puts sum, prod
#=> 15, 120

 

Because Ruby uses :+ and :* as the symbols for the addition and multiplication operations, it's possible to dry that code a little:

arr = [1, 2, 3, 4, 5]
sum = arr.inject(:+)
prod = arr.inject(:*)

puts sum, prod
#=> 15, 120
