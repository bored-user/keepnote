filter

Same thing as JavaScript's filter: takes a block as argument which has access to the current iteration item of the array. If the block returns true, it'll add the element to a new array. If it returns false, it won't. The method returns the formed array.

 

arr = [10, 11, 20, 22, 30, 33, 40, 44, 50, 55]

puts arr.filter {|n| n % 5 == 0}
#=> [10, 20, 30, 40, 50, 55]
