Hashes

Hashes, also known as dictionaries, are key-value pairs, just like Arrays. A hash key, however, can contain any character, not only numbers.

Therefore, hashes are like associative arrays.

# defining a hash
hash = {'key' => 'value'}
puts hash['key'] #=> 'value'

# hashes also can be multi-dimensional:
hash = {
    'hash' => {
        'hash' => 'Example of a 2D hash'
    }
}
puts hash['hash']['hash'] #=> 'Example of a 2D hash'

# hashes have no limitation with data types:
hash = {
    'nil' => nil,
    'true' => true,
    :symbol => 'symbol',
    'number' => 1
}

# it's also possible to set hash values using
hash['float'] = 0.1

# finally, there's a shortcut to set hashes where the key's symbol:
hash = {
    symbol: 'value' #=> same as :symbol => 'value'
}
