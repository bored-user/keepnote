Freezing string

One can freeze a string using the freeze method.

After a string is freezed, it cannot be modified (Ruby will throw an error if one tries to modify the string). This don't mean one cannot set another value to the variable holding the pointer to that string in memory. It only means its not possible to modify it using string[0] = ‘chars’.

Freezing strings is useful because it reduces the memory usage for that specific string.

Ruby provides the magic freeze-string-literal comment that automatically freezes all strings.
