Multi-line

To build a multi-line string, one can use the <<EOM operator or using a double quote block, indicating line breaks with \n sequences.

default_multiline = "Hello,\nWorld!"
eom_multiline = <<EOM
    my very
    long
    string
EOM

puts default_multiline
#=> 'Hello,
#   World!'

puts eom_multiline
#=> 'my very
#   long
#   string'
