Multi-line

There are basically three ways of adding multi line comments in Ruby:

  1. Using =begin and =end blocks
  2. Using strings
  3. After __END__.
=begin
    this is a multi line comment. it won't be read by Ruby parser! it's kind of the default way of doing multi line comments.
=end

"this is a comment string. it's bad because it generates a string instance in memory"

__END__

this won't be parsed
