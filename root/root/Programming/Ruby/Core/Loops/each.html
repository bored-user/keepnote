each

Ruby also has a method named each, which acts like a forEach loop in JavaScript. It can be run on Enumerables & parses its code from a given Block.

arr = [0, 1, 2, 3, 4, 5]
arr.each { |i| puts i } #=> 0 1 2 3 4 5
arr.each do |i|
	puts i #=> 0 1 2 3 4 5
end

 

Returns an Enumerator object.

 arr = %w[a b c]
puts arr.each #=> #<Enumerator: ["a", "b", "c"]:each>

arr.each {|n| puts n} #=> 'a' 'b' 'c'

 

Note that it's also possible to give an explicit block-symbol-method when calling each on an Enumerator:

 arr = %w[q w e r t]
puts arr.map.each(&:upcase)
#=> ["Q", "W", "E", "R", "T", "Y"]

arr = [1, 2, 3]
puts arr.map.each(&:next)

=begin
    `map` could be replaced by `to_enum`.
    One needs to convert the array into an Enumerator to be able to use the block-symbol-method thingy.
=end
