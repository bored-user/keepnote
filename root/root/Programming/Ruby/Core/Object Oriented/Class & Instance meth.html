Class & Instance meth

When creating a class method, one can use the def my_method syntax or def self.my_method syntax. The difference between the two is:

 

  • The first (def my_method) creates an instance method. It's only possible to access it instantiating the class (MyClass.new).

 

  • The second (def self.my_method) creates a class method. Meaning that there's no use to instantiate a class to use it; it's possible through the MyClass.my_method syntax.

 

Finally, a singleton method is a method defined only for a particular object:

class Klazz
	def m1
		'm1'
	end
end

k1 = Klazz.new
def k1.m2
	'm2'
end
k1.m1 #=> 'm1'
k1.m2 #=> 'm2'

k2 - Klazz.new
k2.m2 #=> NoMethodError

 

The same behavior can be said regarding Modules.
