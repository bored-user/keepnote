Access control

Ruby supports the default levels of control in class methods:

  • public
  • protected
  • private

In order to set such access controls, one can either do:

class MyClass
    protected
    
    def my_method_1
        'this is protected!'
    end
    
    def my_method_2
        'this is also protected!'
    end
    
    private
    
    def my_method_3
        'I guess you understood my point'
    end
end 

or 

class MyClass
    def my_method_1
        'this is protected!'
    end
    
    def my_method_2
        'this is also protected!'
    end
    
    def my_method_3
        'this is private!'
    end
    
    protected :my_method_1, :my_method_2
    private :my_method_3
end
