Inherited constructors

When working with inheritance, one will realize that a class automatically inherits the constructor class from the super (or parent) class. It's, however, possible to overwrite the constructor and even call the super class constructor from within the child class, like so:

Â 

class SuperClass
	def initialize
		puts 'Initializing super class'
	end
end

class ChildClass < SuperClass
	# if no constructor is set, it'll use the parent's
	def initialize
		super # calling parent's constructor
		puts 'Initializing child class'
	end
end

child_class = ChildClass.new
#=> 'Initializing super class' 'Initializing child class'
