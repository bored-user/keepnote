attr methods

The attr methods are a way to easily make attributes. They automatically creates the getters and setters functions.

There are three different types:

  • attr_reader
  • attr_writer
  • attr_accessor

The first one is used when one wants to create only the method which returns the instance variable equivalent to its name (basically, only read it).

The second one is used when one wants to create only the name= method, which allows the programmer to modify the attribute value (but not read it).

The third one is a combination of the other two. It allows one to read and write the attribute value.

The syntax is the same for all three: one calls the method giving a Symbol with the attribute name as argument. It's possible to give more than one argument.

class MyClass
	attr_reader :reading
	attr_writer :writting
	attr_accessor :both
	
	def initialize(r, w, b)
		@reading = r
		@writting = w
		@both = b
	end
end


my_class = MyClass.new('you can only read me!', '', 'try reading/ writting me!')

puts my_class.reading #=> 'you can only read me!'
my_class.reading = 'a' #=> No Method ERROR

puts my_class.writting #=> No Method Error
my_class.writting = 'writting...'

puts my_class.both #=> 'try reading/ writting me!'
my_class.both = 'hello, world!'
puts my_class.both #=> 'hello, world!'
