Arrays

Arrays are a variable set. They can be either a nominated list or a numerated list. Arrays (unless set otherwise) ALWAYS start at 0. Arrays are defined using the built-in array function.

$myArray = array(1, 2, 3, 4, 5)
echo $myArray[0] => prints 1

 

$arr = array(
	1 => "val 1",
	2 => "val 2",
	3 => "val 3"
)

echo $arr[1] // prints 1
