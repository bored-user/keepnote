mysqli_stmt_bind_result()

Binds variables to a prepared statement for result storage. It takes $stmt as parameter 1 and variables with the column result as other parameters. It should be called after mysqli_stmt_execute() but before mysqli_stmt_fetch().

$sql = "SELECT id, name FROM `table`";
$stmt = mysqli_stmt_init()
mysqli_stmt_prepare($conn, $sql);
mysqli_stmt_execute($stmt);
mysqli_bind_result($stmt, $col1, $col2);
