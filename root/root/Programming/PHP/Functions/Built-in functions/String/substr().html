substr()

substr() returns a part of a string. It takes three parameters (2 are mandatory): the string, the start position and lenght. The string is the input string. The one to be searched on. The start is where the search will start (positive: left to right; negative: right to left). Length: A "to" parameter. Meaning: get characters from start position to lenght one. If omitted, the substring starting from start until the end of the string will be returned.

Examples:
$string = "abcd"
substr($string, 0) // "abcd"
substr($string, -1) // "d"
substr($string, 0, 1) // "a"
substr($string, 0, -1) // "abc"
substr($string, -2, -1) // "c"
