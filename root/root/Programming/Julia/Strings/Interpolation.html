Interpolation

Inside a string block, one can use the dollar sign to interpolate variables.

Example;
name = "Joao"
println("Hello, $name") # "Hello, Joao"

It's also possible to use parenthesis after the dollar sign to perform arithimetic operations, such as:
num1 = 10
num2 = 10
println("$num1 + $num2 = $(num1 + num2)") # "10 + 10 = 20"
println("$num1 * $num2 = $(num1 * num2)") # "10 * 10 = 100"
