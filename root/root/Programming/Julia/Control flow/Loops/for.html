for

The for loop also executes instructions until a condition is matched. Using it, however, the programmer can iterate directly on the array, instead of on the array keys. It also ends with the end keyword.

for n in 1:3
	println(n) # 1 2 3
end

arr = ['a', 'b', 'c']
for n in arr
	println(n) # 'a' 'b' 'c'
end

arr = [1, 2, 3]
for n in 1:length(arr)
	println(arr[n]) # 1 2 3
end




One can use the equal sign or the "this is an element of" math-sign instead of the "in" keyword.


Finally, it's also possible to condense a nested for loop using

for i in 1:10, j in 1:10
	println(i, j) # 11, 12... 21, 22... 1010
end
