pop!()

Removes a element (identified by the key) from a data structure. Returns the removed element. Takes, as 1st parameter, the data structure and, as (optional) 2nd, the key to remove. If there's no second parameter (key), it'll remove the last element.
ATTENTION: it modifies the data structure.

dict1 = Dict("key1" => "value1", "key2" => "value2")
println(dict1) # has both "key1" and "key2" as entries.
pop!(dict1, "key2")
println(dict1) # has only "key1"
