Train and evaluate on the training set

A linear regression model can be created using sklearn.linear_model.LinearRegression(). Then, another pipeline can be created with this last estimator (a predictor): lin_reg = make_pipeline(preprocessing, LinearRegression()). That takes the result of the previous preprocessing pipeline and feed it to the linear regression model. Finally, lin_reg.fit_transform(X, y) - in which X are the instances and y are the labels.

To measure the result, you can use sklearn.metrics.mean_squared_error.

Another model that can be used is a sklearn.tree.DecisionTreeRegressor: tree_reg = make_pipeline(preprocessing, DecisionTreeRegressor(random_state=42)).

Finally, one can use a Random Forest regressor (from sklearn.ensemble.RamdomForestRegressor). The code is the same as the DecisionTreeRegressor.
