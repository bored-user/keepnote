Look for correlations

A correlation matrix can be used to look for linear correlations on the data. That can be achieved using the df.corr() method.

Values near 1 means strong positive correlation; values near negative 1 means strong negative correlation and values near 0 means no linear correlation.

df.corr()["median_house_value"] returns the correlations on the median house value attribute. It's also possible to sort it using the .sort_values() method.
