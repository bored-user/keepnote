Feature scaling and transformation

There are two main ways to scale data: min-max scaling (normalization) and standardization.

Note that only numerical data can be scaled, so run a housing_num = housing.select_dtypes(include=[np.number]) in order to select only the numerical attributes from all the housing dataset.

The first can be achieved via sklearn.preprocessing.MinMaxScaler, with a feature_range attribute set to a (-1, 1) tuple. Then, the result is outputed by scaler.fit_transform(housing_num).

The second is done by sklearn.preprocessing.StandardScaler(), and then outputed by scaler.fit_transform() as well.
