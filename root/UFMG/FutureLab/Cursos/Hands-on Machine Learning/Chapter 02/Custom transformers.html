Custom transformers

For transformations that don't require training, you can create your own transformers using sklearn.preprocessing.FunctionTransformer class. For instance, log_transformer = FunctionTransformer(np.log, inverse_func=np.exp) creates a transformer which outputs the log of the attribute. Then, you can call log_transformer.transform() to transform the data. Also, log_transformer.inverse_transform() to return the original.
