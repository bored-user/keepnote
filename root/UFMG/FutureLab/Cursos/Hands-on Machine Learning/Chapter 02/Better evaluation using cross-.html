Better evaluation using cross-validation

Using sklearn.model_selection.cross_val_score, you can execute k-fold validation (split the training set into k folds and using the k-1 folds for training and the last one for evaluation. This helps in avoiding overfitting).

tree_rmse = -cross_val_score(tree_reg, housing, housing_labels, scoring="neg_root_mean_squared_error", cv=10)
