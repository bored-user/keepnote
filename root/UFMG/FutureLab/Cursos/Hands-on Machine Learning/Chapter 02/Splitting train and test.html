Splitting train and test

Sklearn has a built-in method to split the train set from the test set:

sklearn.model_selection.train_test_split().

One can specify a test_size float argument and a random_state integer argument - to set the size of the test set (usually 20%) and the random state in order for the test set be equal on every execution of the script.

It is possible to create a stratified sampling using the stratify argument. It takes a column of the DataFrame object and stratifies the data based on it.

Finally, it's possible to create a column on a DataFrame object using Pandas's pd.cut() method. It creates a new attribute on the DataFrame based on a given pre-existing attribute, like so:

pd.cut(df["oldcol"], bins=[0, 1.5, 3, 4.5, 6, np.inf], labels=[1,2,3,4,5]) creates a new DataFrame column based on the column of the given df object. Its ranges work as so: the first category ranges from 0 to 1.5; the second, from 1.5 to 3 and so on. After using, it may be dropped by train_set.drop("newcat", axis=1, inplace=True). The inplace argument modifies the original object; if it's not given to the function, drop() will return a copy of the object.
