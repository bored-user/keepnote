Visualizing the data

Before running tools to visualize what the data looks like, it's usually a good idea to copy it: view = df.copy().

view.plot(kind='scatter', x='longitude', y='latitude', grid=True, s=view["population"] / 100, label="population", c="median_house_value", cmap="jet", colorbar=True)

That creates a scatter plot based on longitude (x) and latitude (y); in which the radius of each circle is the district's population and the color is the meadian house value.
