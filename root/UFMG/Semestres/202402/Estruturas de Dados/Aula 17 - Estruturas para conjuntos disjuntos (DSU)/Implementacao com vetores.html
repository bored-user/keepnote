Implementacao com vetores

Nessa implementacao, cria-se um vetor estatico de n elementos e utiliza-se representantes para montar os conjuntos. Dois elementos estao em um mesmo conjunto se possuem representantes iguais.

Inicialmente, a funcao make inicializa o vetor estatico, fazendo cada elemento ser seu proprio representante (conjuntos[x] = x), sendo que os dados a serem armazenados, x, sao numeros sequenciais de 0 a n. A complexidade da funcao eh O(1).

Para encontrar um elemento, basta retornar o conjuntos[x], ou seja, O(1) tambem.

A uniao encontra o representante dos dois elementos e percorre o vetor inteiro, trocando os representantes de acordo com a necessidade. Portanto, O(n).
