Ordenacao por contagem

Quando ja se sabe o range do vetor (por exemplo, todos os elementos sao inteiros positivos entre 0 e max, max ja conhecido).

A ordenacao eh feita criando um array auxiliar de tamanho max que armazena o numero de vezes que cada elemento do vetor apareceu. Apos isso, esse array de contagem eh percorrido e os itens sao reordenados no array original.

void countingsort(int* values, int n, int max) {
	int* counts = (int*)calloc(max, sizeof(int));
	int i = 0;
	for(i = 0; i < n; i++)
		counts[values[i]]++;
	
	i = 0;
	for(int j = 0; j < max; j++) {
		while(counts[j] > 0) {
			values[i] = j;
			counts[j]--;
			i++;
		}
	}
	
	free(counts);
}

A ordenacao eh feita em O(n); porem, usa muita memoria extra. Alem disso, o metodo so eh viavel se soubermos a faixa de elementos anteriormente.
