Bucket sort

O bucket sort eh uma generalizacao do counting sort, no qual o numero de baldes eh variavel (nao eh sempre o maior elemento do vetor).

Consiste em separar os elementos em baldes de tamanho menor e ordenar separadamente cada um dos baldes utilizando algoritmos tradicionais de ordenacao. Alternativamente, pode-se implementar os baldes como listas encadeadas e ja inserir de forma ordenada em cada lista.

Usa-se O(n/k) para inserir os n elementos em k baldes. Logo, o pior caso eh O(n^2). O Melhor caso eh O(n). Usa-se O(n) memoria extra. Uma das vantagens eh que quando k cresce, a ordenacao eh quase linear: n^2/k → n quando k → n.
