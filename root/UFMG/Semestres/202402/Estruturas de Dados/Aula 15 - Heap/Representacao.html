Representacao

Uma forma de representar o heap na memoria eh usando um vetor estatico de tamanho n (n nos no heap). A ordem em que os nos aparecem no vetor eh a resultante de um caminhamento por nivel.

Dessa forma, se um no aparece na posicao i do vetor, seus sucessores da esquerda aparecerao em 2i+1 e, da direita, em 2i+2. Ou seja:

int GetAncestral(int i) { return (i-1)/2; }
int GetSuccessorEsq(int i) { return 2*i+1; }
int GetSucessorDir(int i) { return 2*i+2; }
