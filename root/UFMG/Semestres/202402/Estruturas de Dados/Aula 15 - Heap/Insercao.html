Insercao

A insercao no heap possui duas partes: primeiro, o novo elemento eh inserido como uma folha da arvore. Apos isso, eh necessario verificar se a propriedade do heap se manteve e conserta-la se necessario (heapify). Seja t a quantidade de elementos do vetor e x o elemento a ser inserido.

Insere(x) {
	v[t] = x;
	i = t;			// comeca do fim
	p = (i-1)/2;	// pai de i
	while v[i] > v[p] {
		swap v[i] v[p];
		i = p;
		p = (i-1)/2; // vai para o proximo pai
	}
	t++;
}

Considerando um heap com n elementos, a insercao eh O(log n). Isso so eh garantido pois, pela forma como a insercao acontece, a profundidade das folhas difere em no maximo 1. Ou seja, a arvore sempre estara balanceada.
