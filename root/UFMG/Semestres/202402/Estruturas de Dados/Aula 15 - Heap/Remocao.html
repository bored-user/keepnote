Remocao

A remocao sempre acontece na raiz da arvore - ou seja, a remocao sempre retorna o maior elemento. Apos isso, eh necessario consertar a arvore caso necessario. O candidato a nova raiz sera sempre o ultimo elemento escrito no vetor.

Remove() {
	x = v[0];
	v[0] = v[t-1];
	t--;
	i = 0;
	s = index maior sucessor de i;
	while v[i] < v[s] {
		swap v[i] v[s];
		i = s;
		s = index maior sucessor de i;
	}
	
	return x;
}

Novamente, a complexidade de tempo eh O(log n).
