Posiciona

Funcao auxiliar para posicionar um ponteiro em uma determinada celula da lista. Usa um booleano para parar na celula anterior ou nao.

Item* Posiciona(int pos, bool antes=false) {
	Item* p;
	
	p = primeiro;
	for(int i = 1; i < pos; i++) p = p->prox;
	
	if(!antes) p = p->prox;
	
	return p;
}

O(1) ou O(n)
