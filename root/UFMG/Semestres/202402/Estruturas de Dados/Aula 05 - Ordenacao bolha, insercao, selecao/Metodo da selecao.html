Metodo da selecao

Selecao do n-esimo menor elemento do vetor (dentre os restantes) e troca entre ele a a n-esima posicao da lista. Apenas uma troca por vez eh realizada.

void Selecao(int* vet, int sz) {
	for(int i = 0; i < sz-1; i++) {
		int min = i;
		for(int j = i+1; j < sz; j++) {
			if(vet[j] < vet[min]) min = j;
		}
		troca(vet[i], vet[min]);
	}
}

O metodo da selecao nao eh estavel, em funcao de um efeito colateral: se vet = {5, 2, 5, 1, 4}, na primeira iteracao (i=0), min valera 3 (indice do elemento “1”). Entao, havera a troca entre vet[i] = vet[0] = 5 e vet[min] = 1. No entanto, nessa troca, a ordem relativa entre os 5's sera invertida.

  • comparacoes: O(n^2) - mesmo raciocinio do metodo da bolha
  • movimentacoes: 3(n-1) = O(n) - sendo o 3 dependente da implementacao da funcao swap

Apesar de ser linear para o numero de movimentacoes, nao eh estavel e nem adaptavel.
