Criterios de classificacao

  • Localização dos dados
    • interna: chaves na memoria principal
    • externa: chaves na memoria secundaria
  • Estabilidade: elementos “duplicados” (ie, manutencao da ordem relativa entre chaves de mesmo valor)
    • estável: quando a ordem relativa dos registros com mesma chave não se altera
    • instavel: quando a ordem relativa se altera
  • Adaptabilidade
    • adaptavel: sequencia de operacoes muda conforme a entrada (nao ordena array ordenado previamente)
    • nao-adaptavel: sempre as mesmas operacoes
  • Uso da memoria
    • in-place: transforma os dados usando espaco extra de tamanho constante (ie, sem array auxiliar), se usa memoria extra ≤ c*log n, sendo n o tamanho da entrada
  • Movimentacao dos dados
    • direta: registro todo eh acessado e movido
    • indireta: apenas os ponteiros sao rearranjados

Metricas de complexidade:

  • numero de comparacoes C(n) - eg, numero de if's
  • numero de movimentacoes M(n) - eg, numero de swaps
