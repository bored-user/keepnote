Matriz de adjacencia

  • Grafo com n vertices. Matriz de adjacencia Anxn, A[i][j] = 1 se existe aresta ij, 0 caso contrario.
  • Como o grafo eh nao dirigido (ie, as arestas nao tem direcao, ie, aresta que conecta i a j conecta tambem j a i), a matriz eh simetrica.
  • Para verificar a existencia de uma aresta, basta verificar o valor da matriz na posicao ij. Logo, O(1)
  • Para verificar os vizinhos de um vertice, basta varrer a linha/ coluna da matriz de adjacencia (ie, achar os vizinhos do vertice 2, basta varrer A[2][i] ou A[i][2]). Melhor utilizar as linhas pela localidade de referencia. Isso eh theta(n).

Vantagens:

  • acesso rapido as arestas, O(1)

Desvantagens:

  • consultar vizinhanca eh theta(n) sempre
  • espaco O(n^2)
