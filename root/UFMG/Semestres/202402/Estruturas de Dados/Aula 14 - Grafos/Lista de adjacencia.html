Lista de adjacencia

  • Uma lista para cada vertice (lista de listas/ vetor de listas) e nessa lista tem os vizinhos daquele vertice.
  • Para verificar a existencia do vertice k, basta percorrer a lista da k-esima entrada da lista de adjacencia, O(n).
  • Para encontrar quem sao os vizinhos de um vertice, basta encontrar a lista referente ao vertice estudado, O(n)

Vantagens:

  • Espaco O(|E|+|V|). |E| = O(n^2), entao no pior caso a lista ocupa espaco proporcional ao da matriz de adjacencia
  • Tempos de execucao menores quando o grafo possui poucas arestas (apesar de o comportamento assintotico ser o mesmo da matriz de adjacencia; constantes importam!)
