Insercao

Metodo recursivo: compara o item com o elemento da raiz

  • se for menor, insere na esquerda
  • se for maior, insere na direita
  • se raiz == null, insere item
void Insere(Item item) {
	InsereRecursivo(raiz, item);
}

void InsereRecursivo(No* &p, Item item) {
	if(p == NULL) {
		p = new No();
		p->item = item;
	} else {
		if(item < p->item) InsereRecursivo(p->esq, item);
		else			   InsereRecursivo(p->dir, item);
	}
}

Pior caso: arvore binaria de maior altura: quando todos os nos da arvore tem apenas um filho (ex.: insercao de elementos em ordem crescente). Nesse caso, a arvore se degenera em uma lista encadeada. O(n)

Melhor caso: arvore binaria de menor altura: quando eh balanceada: todos os nos possuem dois filhos. O(log n)
