fill()

Colors the last drawn rectangle. Can take a single parameter, indicating the greyscale value (0 to 255) or 3 arguments indicating RGB/ HSB (depending on the current colorMode()) or a string with the color name or hex code or rgb(r,g,b) or rgba(r,g,b,a). The default fill is white. When one uses the same color of R, G and B (on an RGB scale), they create what is known by "greyscale color". p5.js anad Processing lets you simlify that on all color functions by just writting it one time. So, if one wants a white background, they should use background(255). Or a black stroke? stroke(0) (which is equivalent to stroke(0,0,0)).
