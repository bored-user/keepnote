functions

Function notation can be used to declare types!

(Double) -> Double // takes and returns a Double
() -> Void // takes and returns nothing
(Int) -> Void // takes Int, returns nothing
() -> String // takes nothing, returns String

 

Therefore, one can set a variable as “type function”:

var func_var: (Int) -> Int

 

And even set said variable to a function:

func sq(n: Int) -> Int {
	return n * n
}
var func_var: (Int) -> Int
func_var = sq

let res = func_var(2) // 4 - arg label isn't used
