One can also use what's known as the loop modifier: the begin/ end loop.

It is basically a loop which has no predefined ending condition.

One can specify any inline loop on the end line, like so:

i = 0
begin
	puts i
	i += 1
end while i <= 10

i = 0
begin
	puts i
	i += 1
end until i >= 10
