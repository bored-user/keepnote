Exception handling

Exception handling in Ruby can be done using begin/rescue blocks.

One can specify the error on the rescue statement or even chain multiple rescues to deal with different errors.

Finally, it's also possible to throw an exception using the raise statement.

begin
	i = 10 / 0
rescue ZeroDivisionError
	puts 'ZeroDivisionError caught!'
rescue
	puts 'Random error caught!'
end

begin
	i = 12
	raise 'MyCustomError'
rescue ZeroDivisionError
	puts 'ZeroDivisionError caught!'
rescue MyCustomError
	puts 'MyCustomEror error caught!'
rescue
	puts 'Other error!'
end
