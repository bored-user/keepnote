Switch

There's also the switch statement, to "join" multiple if-elsif-else blocks:

age = 18
case age
	when 2
		puts 'Age = 2'
		break # this will exit the switch statement
	when 3
		puts 'Age = 3'
	else # just like `default` in other languages
		puts 'I don't know your age'
end

It's also possible to check for multiple things at once:

age = 18
case true
	when age > 0 and age < 18
		puts 'Minor'
		break # this will exit the switch statement
	when age > 17
		puts 'Legal age'
	else # just like `default` in other languages
		puts 'Not sure'
end
