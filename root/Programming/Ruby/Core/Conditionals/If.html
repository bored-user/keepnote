If

If blocks start with the if keyword and ends with the end one.

They can be inline (when there's only one command to be executed) or in block (otherwise).

# Using in block form
age = 18
if age > 17
	puts 'Can drink'
end

# Using inline
age = 18
puts 'Can drink' if age > 17

One can also use the else and elsif keywords:

# Better way: `puts "#{bf} is my best friend!"`
bf = 'Friend_1'
if bf = 'Friend_1'
	puts 'Friend_1 is my best friend!'
elsif bf = 'Friend_2'
	puts 'Friend_2 is my best friend!'
else
	puts 'I don't know what my best friend is!'
end
