Other definition methods

There's also another way to define a Hash: using the Hash.new constructor.

It can take one or no arguments.

 

If it takes an argument, it'll default all keys to that given value.

Meaning:

hash = {}
puts hash['a'] #=> ERROR

hash = Hash.new(1)
puts hash['a'], hash['random_string'] #=> 1 1

 

One can also use the Hash class directly:

puts Hash['a', 100, 'b', 200] 
puts Hash[['a', 100], ['b', 200]]
puts Hash['a' => 100, 'b' => 200]

# result: {'a' => 100, 'b' => 200}
