Definition aliases

There are two array definition aliases:

  1. %w
  2. %W

Both are used when an array contain only strings and are the correct way (to define a word only array) according to conventions.

 

%w should be used when the array elements should be surrounded in single quotes, while %w, when in double quotes.

 

name = 'joao'
interpol_arr = %W[#{name}] #=> ['joao']
non_interpol_arr = %w[a b c] #=> ['a', 'b', 'c']
