index

Returns the starting index of the given argument on the caller (String or Array).

str = 'joao'
puts str.index('j') #=> 0
puts str.index('o') #=> 1

arr = [1, 2, 3]
puts arr.index(2) #=> 1

It can also take an optional second parameter indicating the nth index that should be returned (-1 for last). Defaults to 1.

str = 'joao'
puts str.index('o', 2) #=> 3
puts str.index('o', 1) #=> 2
puts str.index('o', -1) #=> 3
