upto

Is called on an Integer, takes another Integer and a block as arguments. Execute the block X times (difference between the given Integer and the callee, or receiver). If the given number is bigger than the number where the method was called, it won't execute the block.

5.upto(10) { |i| puts i } #=> 5 - prints 5 6 7 8 9 10
10.upto(5) { |i| puts i } #=> 10 - prints nothing

 

Can be also called on a String. The “negative repetitions" behavior is the same.

'a'.upto('f') {|s| puts s} #=> 'a' 'b' 'c' 'd' 'e' 'f'
'a8'.upto('b1') {|s| puts s} #=> 'a8' 'a9' 'b0' 'b1'
