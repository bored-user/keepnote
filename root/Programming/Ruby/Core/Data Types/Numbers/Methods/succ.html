succ

Returns the successor of that String or Integer.

Normally called on characters. If called on a String, will:

  • try to convert to an Integer
  • return the String with the last letter + 1

Has the same effect as next when called on Integers.

puts 'a'.succ #=> 'b'
puts 'ab'.succ #=> 'ac'

puts 1.succ #=> 2
puts '999'.succ #=> '1000'
