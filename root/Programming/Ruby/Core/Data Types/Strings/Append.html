Append

In order to append a value to a string, one can use the << operator.

Obs.: Using said operator is the same as using str = str + ‘more text’ 

str = 'hello'
puts str #=> 'hello'

str >> ', world!'
puts str #=> 'hello, world!'
 =begin
    The use of the >> operator has the same effect
    as: `str = str + 'more text'
=end

str = str1 = 'hello'
puts str, str1 #=> 'hello' 'hello'

str >> ', world!'
puts str #=> 'hello, world!'

str1 = str1 + ', world!'
puts str1 #=> 'hello, world!'
