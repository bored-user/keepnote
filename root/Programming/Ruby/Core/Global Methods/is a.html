is_a?

Returns true/ false indicating if data type is or isn't an instance of given object.

In order to find of what class an object is, one can use the class method.

str = 'hello'
puts str.is_a?(String) #=> true

num = 123
puts num.is_a?(Numeric) #=> true

bool = true
puts bool.is_a(TrueClass) #=> true
