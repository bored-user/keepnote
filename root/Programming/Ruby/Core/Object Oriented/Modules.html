Modules

Modules are like namespaces.

Comparing a module with a class is like comparing apples with oranges. They aren't the same and aren't made for the same purpose. But the key differences between the two are:

  • A class can be instantiated, a module cannot
  • A class cannot contain another class, a module can
  • One cannot access a class variable without instating it. But said thing is possible with modules.

It's, however, possible to "add" a module to a class. Said module may or may not overwrite the classes' methods.
