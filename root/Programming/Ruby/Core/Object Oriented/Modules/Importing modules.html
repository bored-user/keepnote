Importing modules

There are three ways to import a module into a class:

  1. include
  2. extend
  3. prepend

 

The major difference between include and extend is that when using include MyModule on MyClass, one can write MyClass.new.module_method, which works perfectly; while when one uses extend MyModule, it's only possible to write MyClass.module_method. 
 

module MyModule
	def greet
		'Hi from MyModule'
	end
end

class IncludeClass
	include MyModule
end

class ExtendClass
	extend MyModule
end

p IncludeClass.new.greet #=> 'Hi from MyModule'

p ExtendClass.new.greet #=> ERROR
p ExtendClass.greet #=> 'Hi from MyModule'
