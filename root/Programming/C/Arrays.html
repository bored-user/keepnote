Arrays

In C, an array is declared using the data type, followed by the identifier and a pair of square brackets: int arr[]. Furthermore, one can declare an array with an unspecified length if it's initialized during declaration: int arr[] = { 1, 2, 3 };. Alternatively, one can declare an array specifying a size and no values - in this case, default values will be used: int arr[3] declares an array of size 3 as {0, 0, 0}.

A multidimensional array can be declared in the same way: int arr[][3] { { 1, 2, 3 }, { 4, 5, 6 } };. Multidimensional array MUST have bounds for all dimensions except the first.
