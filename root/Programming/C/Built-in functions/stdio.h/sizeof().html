sizeof()

sizeof returns the size of the given data type. Note that one can get the size of an array with it:

int arr1[] = {1, 2, 3};
printf("%d\n", sizeof(arr1) / sizeof(int));

char arr2[] = {'a', 'b', 'c'};
printf("%c\n", sizeof(arr2) / sizeof(char))

It's needed to divide by sizeof(int) and sizeof(char) because the sizeof array returns the size of the array multiplied by the space in memory that each items occupies (e.g an int takes 4 bytes; an array of 10 ints takes 4 * 10 = 40 bytes).

 

Also, it's possible to use sizeof as a keyword: printf("%d\n", sizeof int).
