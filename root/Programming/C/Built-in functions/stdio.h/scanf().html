scanf()

Used to take input from stdin. Can be formated like printf(). Also, takes a reference to variables onto which the typed data will be saved:

int digit;
char character;
double dbl;

scanf("%d", &digit);
scanf(" %c", &character);
scanf("%d", &dbl);

 

The whitespace before %c tells the compiler to skip the last linebreak - which would otherwise be saved to the given variable.
