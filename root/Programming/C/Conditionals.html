Conditionals

C supports if-else-else if blocks, as well as switch statements. Note that, in an if block, the variable is tested against 1; meaning that if(a) is the same as if(a == 1).

if(a)
{}
else if(b)
{}
else
{}

switch(a)
{
	case 1:
	case 3:
		break;
		
	case 2:
	case 4:
		break;
		
	case 5:
		break;
		
	default:
		break;
}
