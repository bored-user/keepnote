Functions

There are three main ways to declare a function:

The first one is using the "function" and "end" keywords, like so:

function sayHello(name)
	return "Hello, $(name)!";
end


The second one is an arrow function, just like in JavaScript: sayHello = name -> “Hello, $(name)!”

The third one is just assigning a value to the function name, like declaring a variable: sayHello(name) = return "Hello, $(name)!"

One can overwrite a function of the same type by another of the same type ONLY. For instance, one can overwrite a arrow function with another arrow function with the same name, but not with a named function (type #1).

Functions can be called normally: fn('a', ‘b’, ‘c’…).
