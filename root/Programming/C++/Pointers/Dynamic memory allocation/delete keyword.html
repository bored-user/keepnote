delete keyword

The delete keyword deletes a pointer.

Note that it is in fact safe to execute delete on a pointer that's already nullptr - or rather, that hasn't been initialized. Therefore, there's no need to check if a pointer is not nullptr before calling delete on it.

Also, after deleting a pointer, it is a good practice to reset it to nullptr; otherwise, it is left as a dangling pointer.

The delete keyword may also delete dynamic allocated arrays: delete[] dyn_arr;
