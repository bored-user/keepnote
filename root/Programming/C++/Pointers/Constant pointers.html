Constant pointers

A pointer can be constant if:

  • one cannot change its pointing address
  • one cannot change the value it points to
  • both of the above

 

For the first case, use an int* const ptr = &int_var;, like so:

#include <iostream>

int main() {
	int a = 12;
	int b = 13;
	
	int* const ptr = &a;
	*ptr = 13;		// Works fine
	ptr = &b;		// ERROR: cannot point somewhere else
}

 

For the second case, const int* ptr = &int_var;:

#include <iostream>

int main() {
	int a = 12;
	int b = 13;
	
	const int* ptr = &a;
	*ptr = 13;		// ERROR: cannot redefine (pointed) value
	ptr = &b;		// Works fine
}

 

The third case is a union of the first two:

#include <iostream>

int main() {
	int a = 12;
	int b = 13;
	
	const int* const ptr = &a;
	*ptr = 13;		// ERROR: cannot redefine (pointed) value
	ptr = &b;		// ERROR: cannot point somewhere else
}
