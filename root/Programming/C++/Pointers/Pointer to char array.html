Pointer to char array

A special type of pointer is a pointer to a char array:

#include <iostream>

int main() {

	const char* a {"hello, world!"};
	std::cout << *a << std::endl;
	std::cout << a[0] << std::endl;
	
	// Both will print 'h'

	return 0;
}

Note that a const char* points to the first character of the given string.
