Files

  • The fstream library is used.
  • ifstream infile("in.txt", fstream::in);
  • ifstream outfile("out.txt", fstream::out);
  • outfile << “aaa”; just like with cout
  • infile >> line; just like with cin
  • file.is_open() returns a boolean. Useful with if blocks after opening and before using
  • file.close()
