String streams

The library sstream is used to handle strings as streams of data. Therefore, #include <sstream>.

#include <iostream>
#include <string>
#include <sstream>

using namespace std;

int main() {
	string line;
	while(getline(cin, line)) {
		stringstream info(line);
		string aux;
		while(getline(info, aux, ';')) {
			cout << aux << endl;
		}
	}
}

That block of code reads a line from cin (including spaces, getline()). Then, it creates an instance of stringstream called info, based on the information on the line read. Then, it reads every few characters in info (a stream, just like cin) and passes it along to aux. It's like String.split() in higher level languages. 

With a stringstream, it would also be possible to use the extraction operator (>>), as if it were cin: info >> aux. In this case, the splitting is done using spaces, just like cin.
