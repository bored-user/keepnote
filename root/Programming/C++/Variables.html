Variables

There are three ways to initialize a variable in C++: assignment initialization (int a = 10;), brace initialization (int a{10}) and functional initialization (int a(10)).

Of course, one may also only initialize a variable and don't define it a value: int a;. However, note that this means that a random value will be assigned to the variable.

An empty brace initialization (int a{}) defines the variable with its default type-value; in case of integers, 0.
