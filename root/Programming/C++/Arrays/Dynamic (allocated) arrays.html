Dynamic (allocated) arrays

A dynamic allocated array is an array which size is chosen at runtime. To declare a dynamic allocated array, one uses a pointer: int* arr{ new int[LENGTH] };, which is equivalent to int* arr = new int[LENGTH];. It is also possible to declare it with elements, int* array{ new int[]{ 13, 234, 55, 45 } };.

Since the length of the array is unknown at compile time, it is not possible to use a foreach loop with it (for(int n : arr)). Also, std::size() does not work on dynamic arrays.
