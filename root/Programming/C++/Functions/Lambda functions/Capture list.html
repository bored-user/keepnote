Capture list

The capture list of a lambda function is a way to use variables that are declared outside of the scope of the lambda function. In C++, a lambda only has access to variables declared in its own scope.

Additionally, one can pass in arguments to a capture list through reference or through a copy.

The variables on a lambda function capture list (both the ones passed by reference and by value) are read-only. However, it is possible to add the mutable keyword at the end of a lambda function declaration to allow it to modify the variables on the capture-list: auto lm2 = [&c]() mutable { â€¦ }.

#include <iostream>

int main() {
	int c = 0;

	auto lm1 = [c]() { std::cout << "inner1: " << &c << std::endl; };
	auto lm2 = [&c]() { std::cout << "inner2: " << &c << std::endl; };
	
	std::cout << "outer0: " << &c << std::endl;
	lm1();
	lm2();
}

It is also possible to use [=] and [&] to get access to all external variables - as copies and as references, respectively: auto lm1 = [=]() {}; and auto lm2 = [&]() {};.
