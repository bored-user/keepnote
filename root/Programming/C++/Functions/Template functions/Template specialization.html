Template specialization

Template specialization is a way to declare a version of a function (aka a signature) that will be used by the compiler if the types are compatible. Basically, it's a way to declare a special-case function.

#include <iostream>

template<typename T> T min(T a, T b) {
	if(a < b)	std::cout << a;
	else		std::cout << b;
	
	std::cout << std:endl;
}

template<> int min<int>(int a, int b) {
	std::cout << "in specialized template: " << std::endl;
	
	if(a < b)	std::cout << a;
	else		std::cout << b;
	
	std::cout << std:endl;
}

int main() {
	float a1 = 10.0, b1 = 5.0;
	int a2 = 10, b2 = 5;
	
	min(a1, b1);
	min(a2, b2);
	
	// #=> 5.0
	// #=> in specialized template: 5 
	
	return 0;
}

The compiler will execute the specialized version of min() on the second std:::cout call, because there's a specific min() version declared to be used with ints.
