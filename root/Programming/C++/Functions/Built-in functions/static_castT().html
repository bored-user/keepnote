static_cast<T>()

The static_cast function takes a type T and converts the given argument into that type, returning the result. It is a way to convert a char back to an int, for example:

#include <iostream>

int main() {
	char c = 65;
	std::cout << c << std::endl;					//'A'
	std::cout << static_cast<int>(c) << std::endl;	//65
}
