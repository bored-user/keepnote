Template functions

Template functions are a way to write only one function which gets translated to multiple functions with multiple signatures.

It's useful, for instance, to write a add() method for floats, doubles, longs and integers: the logic will be the same (return a + b;) but C++'s type system would make one write 4 different functions with 4 different signatures. One function template can solve this.

#include <iostream>

template <typename T> T add(T a, T b) {
	return a + b;
}

template <typename T> T max(T a, T b) {
	if(a > b) return a;
	else 	  return b;
}

int main() {
	int a1 = 10, b1 = 5;
	float a2 = 10.5, b2 = 5.5;
	double a3 = 10.5555555, b3 = 5.5555555;
	long a4 = 1000000000, b4 = 55555555555;
	
	int add1 = add(a1, b1);
	float add2 = add(a2, b2);
	double add3 = add(a3, b3);
	long add4 = add(a4, b4);
	
	int max1 = max(a1, b1);
	float max2 = max(a2, b2);
	double max3 = max(a3, b3);
	long max4 = max(a4, b4);
	
	std::cout << add1 << " " << max1 << std::endl;
	std::cout << add2 << " " << max2 << std::endl;
	std::cout << add3 << " " << max3 << std::endl;
	std::cout << add4 << " " << max4 << std::endl;
	
	return 0;
}

The compiler, when generating machine code, generates four versions of the add and max functions (matching the type system). Templating is just easier for the programmer.
