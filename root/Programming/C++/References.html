References

References are somewhat similar to pointers.

#include <iostream>

int main() {
	int a = 12;
	int& ref = a;
	
	std::cout << "a: " << a << std::endl;
	std::cout << "ref: " << ref << std:endl;
	
	a = 15;
	
	std::cout << "a: " << a << std::endl;
	std::cout << "ref: " << ref << std:endl;
	
	ref = 23;
	
	std::cout << "a: " << a << std::endl;
	std::cout << "ref: " << ref << std:endl;
 }

 

The print statements above will print 12, 15 and 23. Note that both the original variable and the reference are modified when either one is modified.

Some nuances of references: they MUST be initialized at declaration (int& ref; throws a compile error); they cannot be reassigned to reference something else and there's no need to perform dereference operations on references.

An int& is kind of like a const int*, without the ugly pointer notation.
