Input and output

One can use std::cout to print things to the screen. Also, std::cin can be useful to read data from the keyboard. However, std::cin stops reading on a space character.

To avoid that behavior, one can use std::getline(), which takes a input stream (std::cin, usually) and a string variable in which to store the read information.

Another useful trick is while(std::cin >> variable) {}, which executes a code block while a data from that specific type is available to be read from the keyboard.
