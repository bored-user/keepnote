Arrays

An array is defined using typename identifier[n] {};. Note that n is only optional if there are values inside of the curly braces - if that's the case, the compiler will deduct the number of elements in the array and allocate memory accordingly. Also, the curly braces are only needed if one wants to initialize the array with values in it - otherwise, one could write typename identifier[10]; to create an array of type typename and 10 elements.

It is not possible to create an (static) array which's size is set at runtime. For that, use dynamic arrays.
