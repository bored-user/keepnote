std::string

std::string is a string wrapper built on top of character arrays.

#include <iostream>
#include <string>

int main() {
	std::string name = "john";
	std::string last = "doe";
	
	std::cout << name << " " << last;
}

 

When using std::string, it is not necessary to worry about memory leaks, because the compiler automatically handles that logic.
