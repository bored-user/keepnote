std::strrchr()

Takes a char array and a char and returns a pointer to the last instance of the given char found on the given char array.

It is possible to use pointer arithmetic and add a 1 at the return pointer of the function in case one wants to ignore the de-facto instance of the given character in the char array.

#include <iostream>
#include <cstring>

int main() {
	std::cout << std::strrchr("/home/user/Downloads", '/') << std::endl;
	// returns and prints '/Downloads'
	
	std::cout << std::strrchr("/home/user/Downloads", '/') + 1 << std::endl;
	// returns and prints (only) 'Downloads'
}
