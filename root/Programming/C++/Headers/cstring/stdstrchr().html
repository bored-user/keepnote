std::strchr()

Takes a character pointer and a character and returns a pointer to the first found instance of the given character in the given character pointer. If no instance is found, it returns a nullptr.

It's a great tool to check if a substring exists within a given string.

#include <iostream>

int main() {
	std::cout << std::strchr("hello", 'l') << std::endl;
	// returns and prints a pointer to "llo"
	
	std::cout << std::strchr("hello", "a") == nullptr << std::endl;
	// returns nullptr and prints 1
}
