std::strcmp(),strncmp()

strcmp() stands for string compare. It returns a positive value if the first argument appears before the second in lexicographical order (the order in which words appear on the dictionary); zero if both arguments are equal and a negative value if the last shows up before the first.

Essentially, it's a good way to test if strings are equal.

 

strncmp(), on the other hand, takes two arguments and an integer which specifies up to which (one-based) index the comparison should be executed.
