std::nothrow

When using dynamic memory allocation with the new keyword, one may use std::nothrow in order to tell the compiler to not throw an exception in case memory gets exhausted: int* p = new(std::nothrow) int[1000000000000];. If there was no std:nothrow, the program would crash because the array is too large.

When using nothrow, the pointer will get nullptr assigned to it.
