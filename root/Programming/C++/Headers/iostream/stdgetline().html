std::getline()

std::getline() is a function that reads an entire line - not just until a space character, but the whole line, until a line-break character.

It takes two arguments: a buffer (normally std::cin) and a variable in which the read data will be stored:

#include <iostream>

int main() {
	std::string name;
	
	std::cout << "Type your name: ";
	std::getline(std::cin, name);
	std::cout << "Welcome, " << name << std::endl;
}
