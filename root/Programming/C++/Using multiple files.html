Using multiple files

In C++, it is a normal practice to split code across multiple files. When compiling, all one needs to do is to indicate the path of all C++ files, in order for the compilation to finish successfully.

Also, it's possible to declare functions in a header file and simply import it using a #include directive.

// main.cpp
#include <iostream>
#include <custommath.h>

int main() {
	int grt = max(10, 20);
	int sml = min(10, 20);
	
	std::cout << grt << std::endl;
	std::cout << sml << std::endl;
}
// custommath.h
int max(int a, int b);
int min(int a, int b);
// custommath.cpp
int max(int a, int b) {
	if(a >= b) return a;
	else	  return b;
}

int min(int a, int b) {
	if(a <= b) return a;
	else	   return b;
}

In order to compile, simply execute > gcc *.cpp. This way, the compiler will compile custom.cpp and main.cpp. It automatically looks for and links header files (custommath.h) correctly.
