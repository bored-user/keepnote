struct

A struct is a way for the user to define their own type.

#include <iostream>
#include <string>

struct Student {
	std::string name;
	std::string address;
	int age;
	
	
	Student(std::string name, int age, std::string addr) {
		this->name = name;
		this->age = age;
		address = addr;
	}
	
	Student(std::string name) {
		this->name = name;
	}
 };

Note:

  • semicolon after struct definition
  • multiple constructors: name of the struct
  • this acts as a pointer to the current structure.
  • this is not needed, not mandatory. One must use it when the struct has a field with the same name as a variable in the current scope.
