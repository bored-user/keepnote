Foreach loop

It's possible to execute for loops in an array in C++ using the following syntax:

#include <iostream>

int main() {
	int nums[] = { 1, 3, 5, 7, 9 };
	for(int i : nums) {
		std::cout << i << std::endl;
	}

	return 0;
}

Also, it could be for(auto i : nums) {}, which would allow the compiler to deduct the type of the variable i based on the type of the nums array
