interface

The interface{} type is the default type, from which all other types inherit. It's like object (in JS, Ruby, C#, etc).

Any type (struct) that implements all methods of an interface is implicitly cast:

type Shape interface{
	perimeter() int
}

type Circle struct{
	radius int
}

type Square struct{
	side int
}

func (c Circle) perimeter() int {
	return 3.1415 * c.radius * c.radius
}

func (s Square) perimeter() int {
	return 4 * s.side
}

// Now, both Circle and Square are of type Shape
