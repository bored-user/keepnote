map

A map is a key-value pair data structure.

One can define it using variable := map[TKey]TValue{ Â } - where TKey is the key data type and TValue the value's. Inside the curly braces, one can define the pairs on a JSON like syntax:

variable := map[int]string {
	1: "a",
	2: "b",
	3: "c",
}

Note the final comma.
