Primitive types

There are four primitive types in Go: strings, numbers, bools and errors.

Â 

A string is a value of 8 bits (but not necessarily encoded in UTF-8). It can be empty but not nil & is immutable.

Numbers include uint8, uint16, uint32, uint64, int8, int16, int32, int64, float32, float64, complex64, complex128, byte (same as unit8) and rune (same as int32).

Booleans, of course, can be only true or false.
