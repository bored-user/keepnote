Variadic parameters

A function may have variadic parameters (aka rest/spread). The variadic parameter must be the last one.

func sum(x ...int) int {
	result = 0
	
	for _, v := range x {
		result += v
	}
	
	return result
}

sum(1, 2, 3, 4, 5, 6, 7, 8)
