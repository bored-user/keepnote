List comprehension

The basic syntax of a list comprehension is: [x * 2 | x ← [1..10]] - which means: for every x from 1 to 10, double it and store it on the list.

One may add a predicate using a comma: [x * 2 | x ← [1..10], x*2 ≥ 12] takes only numbers which, when doubled, are greater than or equal to 12. It is possible to add more than one predicate - they are separated using commas.
