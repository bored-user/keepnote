Defining

Conventions state that an extension method should follow some naming and structural rules:

  1. it should be on a public static class with the Extensions suffix (like StringExtensions)
  2. it should be a static method with at least one argument: this [data type] [argument name].

For example:

public static class StringExtensions
{
	public static string Concat(this string str, string add) => str + add;
}

public static class Int32Extensions
{
	public static int Add(this int n1, int n2) => n1 + n2;
}

class Program
{
	static void Main()
	{
		Console.WriteLine("hello, ".Concat("world!")) // #=> "hello, world!"
		Console.WriteLine(1.Add(10)) // #=> 11
	}
}
