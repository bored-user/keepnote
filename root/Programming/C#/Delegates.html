Delegates

A delegate is a pointer to one or more method(s).

Its syntax is pretty simple: they're defined on class scope, like enums but are more syntactically similar to methods - having a return type and the possibility of taking arguments.

A delegate object can be called (and, as previously stated, can receive parameters). Its syntax is as follows:

class Program
{
	public delegate void MyDelegate();
}

Note that they're like abstract methods in the sense that there's no method body.

 

Finally, be aware that there's no way of directly setting a delegate value, like delegate a = new delegate() or something like that. What one can do is predefine a delegate method and use it as a type, as shown hereor use a built in type, like System.Action or System.Func. It is, however, possible to create a delegate expression (as I like to call it), which looks very much similar to a lambda function, like so:

class Program
{
	static void Main()
	{
		Action act = delegate() { Console.WriteLine("parameter-less delegate"); };
		act();
		
		Action<string> actp = delegate(string p) { Console.WriteLine(p); };
		actp("it's, however, possible to pass in parameters");
		
		Func<string, int, int, int> funcp = delegate(string s, int n1, int n2)
		{
			Console.WriteLine(s);
			return n1 + n2;
		};
		Console.WriteLine(funcp("and even have return values!", 10, 10));
		
		
		/*
			#=> "parameter-less delegate"
			#=> "it's, however, possible to pass in parameters"
			#=> "and even have return values!"
			#=> 20
		*/
	}
}
