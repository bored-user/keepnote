One can create a class that implements generics using the following syntax:

class CustomList<T>
{
	public void Add(T value)
	{}
	
	public T this[int idx]
	{
		get
		{}
	}
}

Note that it's also possible to use more than one generic parameter:

class CustomDictionary<TKey, TValue>
{
	public void Add(TValue value)
	{}
	
	public TValue this[TKey idx]
	{
		get
		{}
	}
}
