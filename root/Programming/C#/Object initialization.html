Object initialization

Object initialization syntax is an easier way to initialize objects, arrays and etc.

They're that { key = value } thing!

Here's how it works:

  • when initializing arrays, one uses { “item1”, “item2", “item3” } syntax, since arrays are sequential
  • when initializing dictionaries, one uses { ["key"] = “item” } syntax, since dictionaries are made of key-value pairs
  • when initializing objects, one uses { Property = “value”} syntax.
