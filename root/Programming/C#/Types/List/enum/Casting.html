Casting

It's also possible to cast the enum value using explicit conversion (and cast the values back to the enum key, using casting as well).

enum Email
{
	Slow = 0,
	Default = 1,
	Fast = 2
}

public static void Main()
{
	Console.WriteLine((int)Email.Fast);
	// #=> 2
	
	Console.WriteLine((Email)2);
	// #=> Fast	
}
