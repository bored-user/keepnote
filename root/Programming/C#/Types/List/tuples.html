tuples

Tuples are defined using the parentheses notation.

One can set them using the type keywords (like string, int, double, etc) or just the good ol' var.

Some examples:

static void Main()
{
	var tuple1 = (1, "string here", 435.54);
	(int, string, double) tuple2 = (1, "string here", 435.54);
};

One can, then, access the tuple's members using tuple1.Item1, tuple1.Item2, etc.

 

Note that all of those examples above were made using what I call associative tuples. One can create named tuples, so to speak, using this syntax:

static void Main()
{
	(int Member1, string Member2) tuple1 = (1, "string here");
}

and access them using tuple1.Member1 and tuple1.Member2.

 

Finally, be aware that method may return tuples: public (int, string) Meth() => (1, “string here”);
