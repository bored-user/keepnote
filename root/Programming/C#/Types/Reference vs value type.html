Reference vs value type

There are two types of information types on C#:

  • Reference types, which store a pointer to their values
  • Value types, which store their values

 

Built-in value types include numeric & floating point types (int, long, decimal, float, double…), bool and char. structs and enums are also value types.

Built-in reference types include string, object and dynamic . classes, interfaces and delegates are also reference types.

All structures are value types (stored on stack), while all classes are reference types (stored on heap).
