Nullable value types

If one needs to set a value to null, they can declare it using T? or Nullable<T> - both using generics.

Â 

T? can be set to everything T can plus the null value, like so:

bool normal = true;
bool? nullable = null;

// `bool boolean = null` throws an exception

One can identify a value as null using the value == null comparison or checking the HasValue property:

bool normal = false;
bool? nullable = null;

System.Console.WriteLine(normal.HasValue) // exception
System.Console.WriteLine(nullable.HasValue) // false
