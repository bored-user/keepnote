Types

C# is a typed language. Therefore, it's necessary to inform the variable types when declaring them.

 

When declaring a floating point number, the C# compiler uses double by default. The same happens with decimal. To force a float/ decimal, one needs to add the f or m suffix to the number:

float numDouble = 3.14;
float numFloat = 3.14f;
float numDecimal = 3.14m;

 

Finally, one can use the var keyword to allow type inference.
