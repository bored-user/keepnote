using Statement

The using statement is useful when dealing with things that implement IDisposable interface, like files.

It ensures that the Dispose() method will be called - even if an exception happens.

The syntax is as follows:

using System;

class Program
{
	static void Main()
	{
		string manyLines=@"This is line one
		This is line two
		Here is line three
		The penultimate line is line four
		This is the final, fifth line.";

		using (var reader = new StringReader(manyLines))
		{
   			string? item;
    			do {
        			item = reader.ReadLine();
        			Console.WriteLine(item);
    			} while(item != null);
		}

	}
}
