#line

The #line directive lets one modify the line number that will be shown on the compiler in case of errors/ warnings. It can optionally modify the file name as well. The syntax is #line [number] [file name] (e.g #line 10 “script”).

Notice that, after a #line directive, the following lines will continue to be incremented normally - unless there's another #line directive.

Finally, be aware that #line default returns the line counting to the “real counting" and #line hidden hides the line from the debugger.
