Creating

One can create a custom attribute simply by creating a class that derives from Attribute base class. Its nomenclature must be NameAttribute - like CoolAttribute (which can then be used as [Cool] or [CoolAttribute]).

In order to make the attribute able to receive arguments, one must create a constructor on the attribute derived class, like so:

using System;

class CoolAttribute : Attribute
{
	public CoolAttribute(string str)
	{
		// when calling the attribute: [CoolAttribute("Argument string!")]
	}
}
