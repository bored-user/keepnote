Func

A Func does pretty much the same as an Action and their syntax is also very similar.

The only difference between a Func and an Action is that its last argument (the only mandatory one) is the type of the return value - since all Funcs must have a return value.

In short:

class Program
{
	static void Main()
	{
		Func<string> a = delegate() { return "no args"; };
		Func<string, string> b = delegate(string s) { return s; };
		
		Console.WriteLine(a());
		Console.WriteLine(b("with args as well!"));
		
		/*
			#=> "no args"
			#=> "with args as well!"
		*/
	}
}
