Constructor

The DateTime object constructor returns a DateTime object. It has multiple overloads, which can take:

  • no arguments - which returns the date of when it was called;
  • ints for year, month and day;
  • int for year, month, day (not null) and a Calendar object;
  • ints for year, month, day, hour, minute and second;
  • ints for year, month, day, hour, minute, second and a DateTime object specifying the kind;
