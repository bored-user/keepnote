Nested namespaces

Finally, note that it's possible to have namespaces inside namespaces, like System.Threading.Thread. The syntax is as follows: namespace MyParentNamespace.MyChildNamespace {} or like so:

namespace MyParentNamespace
{
	namespace MyChildNamespace
	{
		class Program
		{
			static void Main()
			{
				Console.WriteLine("hello, world!");
			}
		}
	}
}
