ref

The ref modifier allows a method to receive a reference of the value and not the value itself, even if the value is a value type. The ref modifier is very similar to a pointer.

Using it, one can replace this

static void Main()
{
	int n = 1;
	n = AddOne(n);
	Console.WriteLine(n); // #=> 2
}

static int AddOne(int n) => ++n;

with this

static void Main()
{
	int n = 1;
	AddOne(n);
	Console.WriteLine(n); // #=> 2
}

static void AddOne(ref int n)
{
	n++;
}
