out

The out modifier is very similar to the ref modifier. They are both used to pass a reference to a variable (a pointer, basically) as argument to a method.

The main difference between them is that the variable to be given to the out modifier may be initialized with the out call, while it has to have been previously initialized when using ref.

One can also discard the result value passing _ as the variable name (that's common usage with the .TryParse() methods).

Examples:

static void Main()
{
	int n;
	AddOneRef(n); // #=> `n` is now 1
	AddOneOut(out int m); // `m` is now 1
	
	char numAsChar = '1';
	Console.WriteLine(int.TryParse(numAsChar, out _)); // #=> true
}

static void AddOneRef(ref int n) => n++;
static void AddOneOut(out int m) => m++;
