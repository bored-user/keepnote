Downcasting

Downcasting is the process of turning a parent/ base class into a child/ derived class.

One can perform it using the following syntax:

class ParentClass
{}

class ChildClass : ParentClass
{}

class Program
{
	static void Main()
	{
		ParentClass pc = new ParentClass;
		ChildClass cc = (ChildClass)pc;
		// Note that, unline upcasting, downcasting requires a explicit casting
	}
}
