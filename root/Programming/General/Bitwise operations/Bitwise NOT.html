Bitwise NOT

Bitwise NOT is the only bitwise operation that takes only one number. It uses the tilde sign and simply inverts every bit:

const a = 10 // 00001010
const b = 5 //  00000101

console.log(~a, ~b)
// 11110101 = -11 
// 11111010 = -6

There's a formula to predict the result: ~x is equal to -(x + 1) (with the exception of -1 and 4294967295, which “bitwise noted” result is 0).
