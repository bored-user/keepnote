Accessing elements

Uses python default bracket notation for 1d arrays. For higher dimensions, use the comma notation.

import numpy as np

a = np.array([1,2,3])
b = np.array([[1,2,3],[4,5,6])
a[0] #=> 1
b[0, 1] #=> 2

Works with negative indices as well.

A colon means “everything”. In the example above, b[0, :] returns [1,2,3], which are the elements of the first row.

You can replace an entire row by b[0, :] = [9, 8, 7] changes the first row of b to [9, 8, 7].

 

Elements on a numpy array can also be accessed via lists. a[[0,1]] returns [1,2].
