Exception handling

Exception handling in Python is done using try/ except and finally blocks - in conjunction with the raise instruction.

except is what other languages call catch. The finally block is only ran if no exception occurs. In order to throw an error, in Python, one raises an exception.

Also, it's possible to catch more than one error using multiple except blocks: 

try:
	5 / 0
except IndexError as e:
	print('index error: ', e)
except ZeroDivisionError as e:
	print('zero error: ', e)

#=> zero error: division by zero
