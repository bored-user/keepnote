*args and **kwargs

One can pass in an unspecified number of arguments using *args and **kwargs. They work roughly like JS's rest operator. The difference is that *args take in unnamed arguments - while **kwargs take in named arguments. In summary, **kwargs provides a dictionary in which the key is the name of the argument and the value is its value - while *args returns an array of values only.

def fun1(*args):
	for arg in args:
		print(arg, '\n')
		
def fun2(**kwargs):
	for key, val in kwargs.items():
		print('key: ', key, '\n', 'val: ', val, '\n\n')
		
fun1(1, 2, "hello")
"""
	1
	2
	"hello"
"""

fun2(a=1, b=2, c="hello")
"""
	a: 1
	b: 2
	c: "hello"
"""
