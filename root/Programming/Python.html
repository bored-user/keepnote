Python

Python is an interpreted language, ran by the python command. One can also specify the shebang (#!/bin/python), which sets the path to Python's interpreter. If that's done, one doesn't need to call the python command on the terminal - it's possible to simply execute ./script.py and the terminal will implicitly invoke /bin/python. 

Indentation is essential in the language. It does not make use of curly braces, but the interpreter can only run Python code if it's correctly indented - indicating which commands are part of which structure.
