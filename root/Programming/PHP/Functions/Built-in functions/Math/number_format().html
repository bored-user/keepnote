number_format()

The number_format() function formats a number.
It takes 3 optional parameters and a mandatory one:

  • The first parameter (mandatory) is the number.
  • The second is the decimal number quantity.
  • The third is the decimal indicator.
  • The fourth is the thousands indicator.
     

If only the first parameter is given, the function formats the number according to the English notation (3 decimal places, "." for decimals and "," for thousands). If some parameter isn't given it is set to the default one(s).

$number = 123.456
number_format($number, 1, ",") // 123,4
