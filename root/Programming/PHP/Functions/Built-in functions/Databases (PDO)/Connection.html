Connection

In order to connect to a database using PDO statements, one must use:

$pdo = new PDO("mysql:host=[host_name];dbname=[database_name]", $username, $password);
It's also good to use $pdo->setAttribute(PDO::ATTR_DEFAULT_FETCH_MODE, PDO::FETCH_ASSOC);

And return the $pdo variable, which will later be used to query the database.
