mysqli_stmt_bind_param()

Binds the parameters on the original SQL query. Takes 3 or more parameters: $stmt, a string indicating data type ("s" for string, "i" for integer, "b" for BLOB and "d" for double) and the replacement data. Data replacement points are indicated using the question mark.
Returns a boolean indicating success.

Example #1:
$userInput = 123;
$sql = "SELECT * FROM `table` WHERE id = '?'";
$stmt = mysqli_stmt_init($conn);
mysqli_stmt_prepare($stmt, $sql);
mysqli_stmt_bind_param($stmt, "i", $userInput);


Example #2:
$userInput1 = 123;
$userInput2 = "hello";
$sql = "SELECT * FROM `table` WHERE id = '?' AND name = '?'";
$stmt = mysqli_stmt_init($conn);
mysqli_stmt_prepare($stmt, $sql);
mysqli_stmt_bind_param($stmt, "is", $userInput, $userInput2);
