Functions

Defined using function fun($arg1, $arg2) {}. One can call a function using a string:

function fun($arg1) {
	echo $arg1;
}

$var = "fun";

fun("hello, world!");
$var("hello, world!");
// Both print 'hello, world!'

 

Anonymous functions are also supported:

$fun = function($arg1) { echo $arg1; };
$fun("hello, world!"); // #=> hello, world! 

$fun = function() { return "hello, world!"; };
echo $fun(); // #=> hello, world!
