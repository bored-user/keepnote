General explanation

Bitwise operations basically executes the true/ false checks on each bit of the given number.

In order to understand it, one must comprehend binary terms. Roughly speaking, every information is stored on binary (0 or 1). A set of 8 binary numbers (aka bit) is called a byte. A bitwise operation executes the specified operation in all bits of the number.

All bitwise operations (except for bitwise NOT) perform upon an operand and an operator and generate a third number as a result.

Note: 1 = true and 0 = false.
