Constructor inheritance

Constructors are inherited by default. Well, not exactly inherited. They are executed by default before all the code of the child class. If a parent class and the child class constructor takes one argument, the argument given to the child class will be automatically given to the parent class. However, if the number of argument is different, one must use the base methods, passing arguments onto the parent class.

The base method has a similar syntax to constructor overloading:

using System;

class ChildClass : ParentClass
{
	// The base invocation bellow calls the constructor of the parent class.
	public ChildClass() : base()
	{}
}

class ChildClass2 : ParentClass
{
	/*
		It's also possible to pass in arguments given to the child class
		up to the base one.
	*/
	public ChildClass2(string arg1) : base(arg1)
	{}
}
