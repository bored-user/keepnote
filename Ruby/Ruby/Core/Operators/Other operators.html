Other operators

  • Unary operators
    • Unary operators take only one input
    • +'my_string' (creates a mutable copy of string. Same as dup methodon frozen stings)
    • ! (inverts the boolean value of an object)

 

  • * (splat operator)
  • =~ (regex operator)
  • ?: (ternary operator)
  • << (shovel/ push operator - on Strings and Arrays)
  • :: (namespace resolution operator)
  • &. (safe navigator - or optional chaining, in JavaScript) [Ruby 2.3+]

 

# Safe navigator example

# Without it
puts "User #{user.name} is online!" if user && user.online?

# With it
puts "User #{user.name} is online!" if user&.online
#=> Will stop chaining if the object is nil.
