print

print also logs to the console. It doesn't add a line break after the printed data & renders nil to blank space (since it's absence of value).

print 'hello, '
print 'world'
#=> 'hello, ' 'world'

print nil #=>

=begin
    Using `puts 'hello', 'world'` is the same as
    `print "hello\n", 'world'`.
=end
