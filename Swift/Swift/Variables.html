Variables

It's a good practice to declare variables using its type, but it isn't necessary. The var keyword is used to declare variables for the first time (from the 2nd on, its not necessary).

When one doesn't assign a value to a variable at initialisation time, a “default” is declared depending on said variable type.

A variable can change both value and type on new assignment.

 

var isMale : Bool // Declared with `false` value.
var age : Int // 0
var error_var // Error: no type & no value.

var str = "String"
str = "Other string"

uninit_var = 123 // Error: unresolved identifier

var type_value : String = "this var has type & value"
